# Testing Fullstack Backend

A Node.js + Express backend API for authentication and blog management, built with MongoDB and JWT-based security.

This project is part of a fullstack learning application focused on backend architecture, authentication flow, and REST API design.

---

## 🚀 Features

### Authentication
- User registration
- User login
- JWT-based authentication
- Refresh token support 
- Protected routes using middleware verifyJWT

### Blog API
- Create blog posts
- Get all blog posts
- Secure protected endpoints

### Backend Architecture
- Express server setup
- MongoDB integration using Mongoose
- MVC-style structure (controllers, routes, models)
- Custom middleware (logger, JWT verification)
- CORS configuration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- JSON Web Token (JWT)
- Cookie-based authentication 
- CORS middleware
